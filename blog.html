Example: Day 5 - Event Log Analysis

Here's a sample script from the challenge that demonstrates automated security event monitoring:

# PowerShell script for security event monitoring
$EventLog = Get-WinEvent -FilterHashtable @{
    LogName = 'Security'
    ID = 4625  # Failed login attempts
    StartTime = (Get-Date).AddHours(-24)
}

$FailedLogins = $EventLog | ForEach-Object {
    $Message = $_.Message
    $Account = ($Message -split '\n' | Select-String 'Account Name:').ToString().Split(':')[1].Trim()
    $SourceIP = ($Message -split '\n' | Select-String 'Source Network Address:').ToString().Split(':')[1].Trim()
    
    [PSCustomObject]@{
        TimeCreated = $_.TimeCreated
        AccountName = $Account
        SourceIP = $SourceIP
        EventID = $_.Id
    }
}

# Generate security report
$Report = $FailedLogins | Group-Object AccountName | Sort-Object Count -Descending
$Report | Export-Csv -Path "SecurityAlert_$(Get-Date -Format 'yyyyMMdd').csv" -NoTypeInformation

Write-Host "Security analysis complete. Found $($FailedLogins.Count) failed login attempts."

This script demonstrates how PowerShell can automate security monitoring tasks that would otherwise require manual log analysis, directly applying to my work in cybersecurity operations.
